# trussc-addons

TrussC addon registry. Lists community and official addons that can be installed via `trusscli addon clone`.

## How it works

- **`main` branch**: Contains `addons_list.json` — a human-edited list of addon repository URLs.
- **`gh-pages` branch**: Contains `registry.json` — auto-generated by GitHub Actions with version info and descriptions fetched from each addon's `addon.json`.

The `trusscli` CLI fetches `registry.json` from GitHub Pages to discover available addons.

## Adding your addon

1. Optionally, create an `addon.json` in the root of your addon repository. All fields are optional — the addon works without this file, but metadata makes it discoverable:
   ```json
   {
     "description": "Short description of what your addon does",
     "version": "1.0.0",
     "author": "Your Name",
     "license": "MIT",
     "trussc_version": ">=0.4.0",
     "screenshot": "screenshot.png",
     "keywords": ["physics", "2d"]
   }
   ```

   **Notes:**
   - **No `name` field** — the addon name is the repository/folder name (e.g., `tcxMyAddon`).
   - **`version`**: Your self-reported version. If you also use git tags (e.g., `v1.0.0`), the tag is stored separately as `latest_tag` in the registry.
   - **`screenshot`**: Relative path to an image in your repo, or an absolute URL. Used for future addon browser on trussc.org.
   - **`keywords`**: Helps `trusscli addon search` find your addon by topic.

2. Open a pull request adding your addon to `addons_list.json`:
   ```json
   { "name": "tcxMyAddon", "url": "https://github.com/yourname/tcxMyAddon" }
   ```

3. After merge, the GitHub Actions workflow will automatically update `registry.json`.

## Using addons

```bash
# Search available addons
trusscli addon search physics

# List all addons (local + remote)
trusscli addon list --remote

# Clone an addon to your TrussC installation
trusscli addon clone tcxMyAddon

# Add it to your project
trusscli addon add tcxMyAddon

# Update an addon to the latest version
trusscli addon pull tcxMyAddon

# Update all cloned addons
trusscli addon pull --all
```
